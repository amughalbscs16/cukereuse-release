# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as its
code of conduct. The canonical text is maintained by the Contributor
Covenant organisation at:

> <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

All contributors, maintainers, issue reporters, and participants in
project discussions are expected to read and abide by the Contributor
Covenant as linked above.

## Scope

This Code of Conduct applies within all project spaces, and also
applies when an individual is officially representing the project or
its community in public spaces. Examples of representing the project
include using an official project e-mail address, posting via an
official social-media account, or acting as an appointed
representative at an online or offline event.

## Reporting

If you experience or witness behaviour that violates the Contributor
Covenant in the context of this project, please report it privately
to the corresponding author:

> Ali Hassaan Mughal &nbsp;|&nbsp; `alihassaanmughal.work@gmail.com`

All reports will be reviewed and investigated promptly and fairly.
The project team is obligated to respect the privacy and security of
the reporter of any incident.

## Enforcement

Project maintainers are responsible for clarifying and enforcing the
standards set out by the Contributor Covenant. The enforcement
guidelines published at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines>
will be used when determining the consequences of any action deemed
in violation of this Code of Conduct.
